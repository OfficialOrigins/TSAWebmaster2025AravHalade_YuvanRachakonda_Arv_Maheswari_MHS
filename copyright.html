Our Privacy Policy


This Privacy Policy document is part of a student project created for the Technology Student Association (TSA) Webmaster competition. As this is a demonstration website, no real user data is collected or stored. Any forms or interactive elements are for display purposes only and do not process actual information.

While we've implemented this privacy policy to demonstrate best practices in web development, please note that Verdura is a fictional restaurant created solely for educational purposes.



TSA Competition Disclaimer


This website was created for the Technology Student Association (TSA) Webmaster competition and is not associated with a real restaurant or business. Verdura is a fictional establishment designed to showcase web development and design skills for educational purposes only.

Technical Framework & Dependencies


This website was developed using the following frameworks, libraries, and plugins. Note that while some of these were included in the initial setup, not all features were implemented in the final design:


  • Core Frameworks:
    • Bootstrap 4.0 - Frontend framework for responsive design
    • jQuery 2.2.3 - JavaScript library for DOM manipulation

  • Animation & Visual Effects:
    • Animate.min.css - For element animations
    • Owl Carousel - Responsive carousel slider
    • Revolution Slider - Advanced slider plugin (included but not implemented)
    • jQuery Parallax - For parallax scrolling effects

  • UI Components:
    • Font Awesome - Icon library and toolkit
    • Bootsnav - Bootstrap navigation menu
    • Cubeportfolio - Portfolio and grid layouts (included but not implemented)
    • Fancybox - Lightbox gallery implementation (included but not implemented)

  • Additional Features:
    • Google Maps API - For location services (included but not implemented)
    • Final Countdown - For timer implementations (included but not implemented)
    • Kinetic.js - For canvas animations (included but not implemented)

All frameworks and plugins are included in our documentation, though some features were not utilized in the final website. The implemented tools were chosen specifically to create an efficient, responsive, and user-friendly web experience while also following the TSA guidelines.

Framework Implementation Statement


This website uses Bootstrap as a framework system. We affirm that all templates and themes used with this framework were custom-built by our team, with no pre-built templates or themes used.
